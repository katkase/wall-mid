# Wall middleware

![graphql apollo](http://i.imgur.com/oTDU6as.png)

GraphQL + Apollo Server Express

## Versions
- Express 4.16.2
- GraphQL 0.12.3
- Apollo Server Express 1.3.2

## Installation

Install npm dependencies

```
npm install
```

## Development

Start the server

```
npm run dev
```
